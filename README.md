# OpenCV Centroid Tracking
Track faces using centroid tracking algorithm.


## Setup
Generate Virtual environment
```bash
python3 -m venv ./venv
```
Enter environment
```bash
source venv/bin/activate
```
Install required libraries
```bash
pip install -r requirements.txt
```
Run program 
```bash
python3 main.py
```
